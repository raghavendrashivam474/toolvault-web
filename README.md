# ToolVault Web

ToolVault is an engineering infrastructure platform empowering innovators
through access to professional engineering equipment, collaborative workspaces,
expert guidance, and end-to-end prototyping infrastructure.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
