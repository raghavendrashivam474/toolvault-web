# Release Notes — v1.0.0

**Release date:** 2026-07-09
**Codename:** Foundation
**Live at:** https://toolvault-web.vercel.app

---

## Summary

Version 1.0.0 marks the first production-ready release of the ToolVault
public website. Built across 6 sprints, the release delivers a premium
engineering brand experience with full accessibility, SEO, performance
optimization, and validated cross-browser compatibility.

---

## What's Included

- **Homepage** — 10 sections telling ToolVault's story
- **Alternating background rhythm** — dark and light "acts" for cinematic flow
- **Engineering visual identity** — blueprint textures, glass cards, ambient glows
- **Responsive design** — validated from 320px to 1920px
- **Accessibility** — WCAG 2.1 AA, keyboard navigation, reduced motion
- **SEO** — structured data, OG images, sitemap, robots.txt, manifest
- **Performance** — 100 desktop / 96 mobile on PageSpeed Insights
- **Automatic deployment** — GitHub push → Vercel deploy

---

## Sections

1. Hero — dramatic dark introduction with brand statement
